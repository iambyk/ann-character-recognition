# Dataset Documentation

## Character Recognition Dataset

### Generation Process
Characters are manually designed as binary matrices representing pixel grids.

### Font Specifications
- **Regular**: Standard sans-serif representation
- **Italic**: 5° right tilt applied
- **Bold**: Doubled stroke pixels

### Statistics
- Total samples: 15
- Classes: 5 (A, B, C, D, E)
- Features per sample: 35 (7×5 matrix flattened)
- Format: Binary (0/1)

## Logic Gate Dataset
Standard truth tables for binary gates.
